HDU 4252 A Famous City

2023-09-17

题目

A Famous City

题目大意

一张相片上的很多建筑相互遮住了,根据高低不同就在相片上把一座高楼的可见部分作为一个矩形,
并用数字描述其高度,若一张相片上的两个建筑群中间有空地,高度则为0;求最少有多少个建筑;

思路

把每一竖行当作一座建筑然后再排除：高度为0排除，形成矩形排除。
从左往右开始数，若中间没有间隔，一旦出现高度一样的就排除掉

代码

#include<bits/stdc++.h>
using namespace std;
const int N=1e5+9;
int a[N];
void solve()
{
	int n;int j=0;
	while(cin>>n)
	{
		 j++;int ans=n;
		 for(int i=1;i<=n;++i)cin>>a[i];
		 for(int i=1;i<=n;++i)
		 {
		 	if(a[i]==0)ans--;
		 	else{
		 		for(int k=i+1;k<=n;k++)
		 		{
		 			if(a[i]>a[k])break;
		 			if(a[i]==a[k])
		 			{
		 				ans--;
		 				break;
		 			}
		 		}
		 	}
		 }
		 
		cout<<"Case "<<j<<": "<<ans<<'\n';
	}
}
int main()
{
	int t=1;
	while(t--)solve();
	return 0;
}
